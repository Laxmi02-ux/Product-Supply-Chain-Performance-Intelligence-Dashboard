# Product-Supply-Chain-Performance-Intelligence-Dashboard
Project Title: Product &amp; Supply Chain Performance Intelligence Dashboard.

Developed a robust supply chain analytics solution by building a relational SQLite database and connecting it to Power BI via ODBC. Designed a star schema data model linking products, orders, inventory, suppliers, and customer feedback. Implemented DAX measures to evaluate revenue contribution, inventory risk, supplier efficiency, and customer satisfaction. Delivered an interactive BI dashboard that enables stakeholders to monitor supply chain performance and make data-driven decisions in real time.

Key Skills:
Power BI, SQLite, Star Schema, Data Modeling, DAX, Supply Chain Analytics, Business Intelligence.
